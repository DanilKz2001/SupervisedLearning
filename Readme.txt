Из «Бета-Банка» стали уходить клиенты: каждый месяц небольшая часть людей перестает пользоваться услугами банка. Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.

Нужно спрогнозировать, уйдёт клиент из банка в ближайшее время или нет. Предоставлены исторические данные о поведении клиентов и расторжении договоров с банком. 

Постановка задачи: построить модель с предельно большим значением F1-меры, не меньшим 0.59. Проверить F1-меру на тестовой выборке; дополнительно исследовать AUC-ROC метрику.

Этапы исследования:
1. Загрузка данных, предварительная обработка данных. Разведочный анализ данных (EDA).
2. Исследование количественного соотношения классов в выборке и изучение моделей без учета дисбаланса.
3. Борьба с дисбалансом: попытки улучшить качество моделей путем коррекции дисбалансированности исходной выборки.
4. Проведение финального тестирования. Подведение итогов.

Исследуемые в проекте модели машинного обучения: модель логистической регрессии, модель решающего дерева, модель случайного леса.
Для борьбы с дисбалансом классов используются техники взвешивания классов, увеличения выборки (upsampling), а также изменения порога классификации.

Результат исследования и применения моделей.
Наилучшие показатели метрик на валидационной выборке получены при использовании модели случайного леса со взвешиванием классов и коррекцией порога: 
- F1 = 0.654
- AUC-ROC = 0.791
- Recall = 0.689
- Precision = 0.622
- Threshold = 0.504

Гиперпараметры "победившей" модели случайного леса:
- max_depth = 10
- n_estimators = 23
- min_samples_split = 2
- min_samples_leaf = 9
- max_features = 5

На тестовой выборке удалось достичь значения F1 меры, равного 0.619, что больше критического для проекта значения 0.590.

Модель правильно классифицирует 66% объектов положительного класса. При это среди положительных предсказаний верными оказываются больше половины: 58%. (в этой задаче метрика recall важнее; важно охватить как можно больше клиентов, которые могут уйти; и хорошо то, что значение метрики recall уж не меньше, чем precision).

Метрика AUC-ROC для модели составляет 0.79. Модель адекватна - в сравнении со случайным классификатором. Она способна предсказывать уход клиента лучше, чем случайная модель.

Ислледование можно продолжить и, вероятно, улучшить модели при помощи данных маркетологов, которые сравнили стоимости привлечения нового клиента и удержания текущего.
